## 🚀 Overview

**SmartFi** isn't just another expense tracker. It's an **Intelligent Financial Behavior Analysis Engine** designed to understand *how* you spend, not just *what* you spend on. By combining advanced predictive algorithms with an intuitive dashboard, SmartFi helps you identify bad habits, forecast future expenses, and achieve your financial goals faster.

## ✨ Features

- 🧠 **Smart Behavior Analysis**: Discover your spending personality (e.g., "Disciplined Saver" vs "Impulsive Spender") based on your transaction history.
- 🔮 **Predictive Spending**: AI-driven algorithms forecast your end-of-month expenses so you can adjust your budget before it's too late.
- 📄 **Automated Statement Parsing**: Upload your bank statements (CSV/PDF) and watch SmartFi automatically extract and categorize transactions.
- 🎯 **Dynamic Goal Tracking**: Set financial milestones and track your progress in real-time.
- 💸 **Subscription Detection**: Automatically identify recurring payments and hidden subscriptions draining your wealth.
- 💎 **Wealth & Asset Management**: Track the value of your investments including Stocks and Cryptocurrencies alongside your fiat cash.
- 💬 **AI Financial Assistant**: Ask questions about your spending in natural language (e.g., "How much did I spend on food this month?").
- 🚦 **Financial Health Score**: Get an instant grade on your financial wellbeing based on savings ratios and budget adherence.

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Frontend Engine**: HTML5, CSS3, JavaScript (Vanilla JS/Dynamic Templates)
- **Data Processing**: Custom Python Financial Engine (`finance_engine.py`)
- **Storage**: Local JSON database (No cloud dependencies, 100% private)

## 💻 Installation

To run this project locally, you will need Python 3.8+ installed on your system.

```bash
# 1. Clone the repository
git clone https://github.com/krishtiwari30/Smart-Finance-Expense-Tracker.git

# 2. Navigate into the directory
cd Smart-Finance-Expense-Tracker

# 3. (Optional but recommended) Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# 4. Install dependencies (Flask, Werkzeug, etc.)
pip install Flask werkzeug

# 5. Run the application
python app.py
```

## 🎯 Usage

1. Open your browser and navigate to `http://127.0.0.1:5000`
2. **Dashboard**: View your overall financial health score, metrics, and smart alerts.
3. **Transactions**: Add manual transactions or upload your bank statements to auto-populate your ledger.
4. **Analytics**: Dive deep into predictive models and your personalized spending personality report.
5. **Goals & Budgets**: Set category limits and track your progress toward large purchases.
6. **Wealth**: Log your stocks and crypto assets to see your entire net worth in one place.
